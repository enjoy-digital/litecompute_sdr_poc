# [> SDR GUI
------------

## [> Prerequisites
-------------------

`libsdl2`
```bash
sudo apt install libsdl2-dev
```

`imgui`
```bash
git clone https://github.com/ocornut/imgui.git
```

`tiny-process-library`
```bash
https://gitlab.com/eidheim/tiny-process-library
```

## [> Build
------------

```bash
make
```

## [> Run
---------
``bash
./build/sdr_gui
```
